% ========== 摘要 ==========
\begin{center}
    \heiti\large 摘要
\end{center}

\noindent 随着数字化程度不断提高，个人数字资产（加密货币、云账号凭证、加密文件等）在财富结构中的占比持续攀升。然而，这类资产普遍存在持有者去世后永久丢失、平台隐私政策导致继承困难、以及持有人遭受胁迫攻击等严峻安全问题。针对上述挑战，本文提出并实现了一种基于门限密码学与同态承诺的可验证数字资产继承协议——数字遗产管家（Digital Legacy Guardian）。该系统采用 Shamir 秘密共享将主密钥拆分为多份份额，分布式托管给多位监护人，并创新性地引入 Pedersen 同态承诺双多项式结构，在 server 不存储主密钥的前提下实现了可验证的份额分发与零和多项式同态刷新。同时，系统设计了胁迫码机制以抵抗胁迫攻击，支持时间锁与社会共识两种触发模式。实验结果表明，该方案在 3-of-5 门限配置下，份额拆分计算耗时 58ms，同态刷新计算耗时 12ms\footnote{此处为占位数据，需补充实际测试结果}，主密钥恢复正确率 100\%\footnote{需补充实际测试验证}，具备良好的实用性和安全性。与 Ledger Recover、Safe\{Wallet\} 等现有方案相比，本方案在零信任 server、隐私保护和抗胁迫能力方面具有显著优势。

\vspace{0.5em}
